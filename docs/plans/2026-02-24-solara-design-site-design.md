# Solara Design — Дизайн сайта

## Бриф

- **Бренд:** Solara Design (новый бренд, не связан с Curtains World)
- **Ниша:** Шторы на заказ в Дубае
- **Аудитория:** Русскоговорящие в Дубае / ОАЭ
- **Продукты:** Шторы (тюль, блэкаут, день-ночь, римские) + жалюзи/рулонные + моторизация
- **Стек:** Чистый HTML/CSS/JS (статический многостраничный сайт)
- **Стиль:** Светлый минимализм, премиум-позиционирование
- **Формат:** Многостраничный каталог-ориентированный сайт
- **Референсы:** BlindsAndCurtains.ae (структура), FixitCurtains.com (каталог), Antonovich-Design.ae (премиум + RU)

## Структура главной страницы

1. **Header** — логотип, навигация, телефон, WhatsApp
2. **Hero** — полноэкранное фото интерьера, H1 "Шторы на заказ в Дубае", CTA "Бесплатный замер"
3. **Преимущества** — 4 иконки (бесплатный замер, пошив 4-5 дней, установка, гарантия)
4. **Каталог** — 3 карточки-категории (Шторы, Жалюзи, Моторизация) со ссылками
5. **Портфолио** — сетка 6-8 фото работ + ссылка "Смотреть все проекты"
6. **Как мы работаем** — 4 шага (замер → выбор ткани → пошив → установка)
7. **Калькулятор** — встроенный расчёт стоимости
8. **Отзывы** — карусель отзывов клиентов
9. **FAQ** — аккордеон с частыми вопросами (Schema FAQPage)
10. **CTA-блок** — "Закажите бесплатный замер", форма (имя + телефон + WhatsApp)
11. **Footer** — контакты, навигация, соцсети, адрес в Дубае, карта

## Структура страницы продукта

Каждая продуктовая страница (например /shtory/blekaut/) содержит:

1. **Breadcrumbs** — Главная > Шторы > Блэкаут (Schema BreadcrumbList)
2. **H1** — "Блэкаут шторы на заказ в Дубае"
3. **SEO-текст** — 200-300 слов уникального описания
4. **Галерея** — 6-8 фото этого типа штор
5. **Преимущества** — специфические для данного типа
6. **Ткани и цвета** — палитра доступных тканей
7. **Цены** — от ... AED / м²
8. **FAQ** — вопросы по этому типу (Schema FAQPage)
9. **CTA** — заказать замер
10. **Связанные продукты** — ссылки на смежные типы

## SEO-стратегия

| Элемент | Реализация |
|---------|-----------|
| Schema.org | LocalBusiness, Product, FAQPage, BreadcrumbList на каждой странице |
| Meta | Уникальные title + description под каждый кластер запросов |
| Geo-теги | geo.region=AE-DU, geo.placename=Dubai, координаты |
| hreflang | ru как основной, en для будущей EN-версии |
| Open Graph | og:title, og:description, og:image для каждой страницы |
| Sitemap | XML sitemap со всеми страницами |
| Аналитика | Google Tag Manager + Яндекс.Метрика |

## Технические решения

| Аспект | Решение |
|--------|---------|
| Файловая структура | /index.html, /shtory/index.html, /shtory/blekaut/index.html и т.д. |
| CSS | Один styles.css + CSS-переменные для цветовой палитры |
| JS | Минимум: меню, калькулятор, lazy-load, формы |
| Шрифты | Локальные woff2 |
| Изображения | WebP + lazy-loading, srcset для адаптивности |
| Компоненты | Header/Footer через JS-include (без дублирования) |

## Цветовая палитра (светлый минимализм)

- Фон: белый (#FFFFFF) и светло-серый (#F8F8F8)
- Текст: тёмно-серый (#2D2D2D)
- Акцент: тёплый золотистый или бронзовый
- Вторичный: мягкий бежевый (#F5F0EB)
